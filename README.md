# Weatherpy
Summary

A Python script to visualize the weather of 500+ cities across the world of varying distance from the equator.

Requirements:

json

seaborn

requests

matplotlib

numpy

pandas

random

time

citypy


Api Keys:

Openweathermap - https://openweathermap.org/api
